# Luke the Crook

A 2D platformer game built in Unity as a college project, featuring level-based exploration, collectibles, and hazards.

## Gameplay

- Platforming through tile-based levels
- Collectible orbs and interactive doors
- Player health and hazard system
- Win zones to complete levels
- Pause and settings menus

## Controls

- Supports both keyboard input and on-screen touch controls (thumbstick + touch buttons) for cross-platform play

## Built With

- Unity
- C#
- Cinemachine (camera system)

## Running the Game

Download and run `Luke_the_Crook.exe` (Windows), or open the project in Unity to build it yourself.
